Overall Reflection & Learning Journey

The Paradigm Shift to Declarative UI

This course fundamentally transformed how I architect mobile applications. Transitioning from legacy imperative views to Jetpack Compose introduced me to the concept of Unidirectional Data Flow (UDF). Initially, managing state hoistings and recomposition triggers presented a steep learning curve, but it ultimately allowed me to build much more predictable and robust user interfaces.

From Frontend to Full-Stack Integration

The progression from Lab 1 to Project 2 was highly rewarding. Implementing local persistence with Room Database taught me the importance of offline-first capabilities. Later, integrating hardware sensors (CameraX) and remote cloud services (Firebase Firestore & OpenFoodFacts API) bridged the gap between a simple local app and a production-ready, globally connected mobile solution.

In accordance with modern development practices, Generative AI tools (such as ChatGPT/Gemini) were utilized strictly as educational assistants during this project. They were primarily used to help troubleshoot complex Kotlin dependency conflicts, suggest layout optimizations for Jetpack Compose, and generate standard HTML/CSS boilerplate code for this very e-portfolio website. All core business logic and app architectures were independently structured and written by me.
